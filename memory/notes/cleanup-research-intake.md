# CleanUp — Research Intake

Zweck: Sammelstelle für verdichtete Research-Deltas aus vergangenen und neuen Batches.

Regeln:
- keine Rohtextablage im Standardfall
- pro Batch nur kurze Qualitätsbewertung + strukturierte, integrierbare Deltas
- Widersprüche sichtbar markieren, nicht still glätten
- nur bei direkter Projektwirkung zusätzlich einen Hinweis für das Execution Board vorbereiten
- wenige saubere Punkte sind besser als breite Mischzusammenfassungen
