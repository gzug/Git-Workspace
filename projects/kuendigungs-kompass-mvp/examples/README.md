## Zweck

Nutzen für:
- Snapshot-Tests
- Prompt-/Template-Vergleiche

## Hinweis

Wenn die spätere Implementierung stark von diesen Fixtures abweicht, zuerst `RESULT-MAPPING.md` und `GOLDEN-OUTPUTS.md` prüfen — nicht sofort das Schema ändern.
