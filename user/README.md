# User-Quellen

Dieser Ordner trennt kuratierte Nutzerabbilder von rohen Nutzerquellen.

## Struktur
- `../USER.md` → kuratierte, arbeitsrelevante Kurzfassung für die laufende Zusammenarbeit
- `raw/` → persönliche Rohnotizen, Interessenlisten, Ideensammlungen, spätere Fundstücke

## Arbeitsregel
- `raw/` ist **kein Auto-Kontext**
- Inhalte dort nur **gezielt und bei echtem Hebel** lesen
- Relevantes bei Bedarf nach `USER.md`, `MEMORY.md`, `memory/notes/` oder ein Projektartefakt verdichten
- Rohdateien selbst nicht blind in Commits oder operative Projektarbeit ziehen
